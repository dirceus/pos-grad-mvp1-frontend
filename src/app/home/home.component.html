Seja bem-vindo ao APP Base de Questões

Essa aplicação contém os MVPs (Produtos Minímo Viáveis) desenvolvimentos pelo aluno Dirceu Sobrinho do curso de Pós-Graduação em Engenharia de Software da PUC-RIO.


MVP 1 - Disciplina: Desenvolvimento Full Stack Básico


Requisitos:

(Backend) API implementada em Python e Flask com pelo menos 3 rotas (Backend) Fazer o uso de um banco de dados SQLite, PostgreSQL ou MySQL com pelo menos uma tabela (Backend) Documentação da API em Swagger (Frontend) Desenvolvimento de uma SPA (Single Page Application)

Descrição do MVP:

Hipotese a ser validada: API com questões de vestibulares e de concursos seria útil para instituiçoes de ensino e cursos prepatórios Objetivo: Disponibilizar para insitutuições de ensinos uma grande quantidade de questões classificadas por disciplinas e assuntos para que possam ser utilizadas na construção de exercícios, simulados e avaliações. Problema a ser resolvido: Evitar que as insitituições precisem cadastrar e manter uma base de questões Solução> foi desenvolvida uma API e uma aplicação web para cadastro, consulta e exclusão de questões

MVP 2 - Disciplina: Qualidade de Software, Segurança e Sistemas Inteligentes


Requisitos:

(Notebook de Machine Learning) Processo de criação do modelo (Notebook de Machine Learning) Análise de resultados do modelo: (Aplicação Full Stack) Integração do modelo de machine learning com a aplicação full stack (Aplicação Full Stack) Teste automatizado do modelo no back-end da aplicação

Descrição do MVP:

Hipotese a ser validada: Se é possível classificar a disciplina de uma questão a partir de seu enunciado Objetivo: Disciplina de uma questão seja obtida automaticamente com um alto grau de precisão Problema a ser resolvido: Evitar o esforço que um usuário humano precise categorizar uma questão Solução: foi desenvolvimento um modelo utilizando machine learning e intregado a funcionalidade de cadastro de questões do APP Base Questões para determinar a disciplina a partir de seu enunciado.
